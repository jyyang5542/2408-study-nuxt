# 2408-study-nuxt

Vue3 기반의 nuxt.js 프로젝트 탐구

## 01. 프로젝트 세팅

```bash
npm install
```

## 02. 개발 서버 부팅

다음 경로에서 실행됩니다. `http://localhost:3000`

```bash
npm run dev
```

## 03. 산출물

```bash
npm run build
```

로컬 환경에서 산출물 미리 보기:

```bash
npm run preview
```
